# Wishlist

Простая статическая страница wishlist для публикации через GitHub Pages.

## Как опубликовать

1. Запушить проект в репозиторий GitHub.
2. Открыть `Settings` -> `Pages`.
3. В `Source` выбрать `Deploy from a branch`.
4. Выбрать ветку `main` и папку `/ (root)`.

После этого сайт будет доступен по адресу:

`https://applemause.github.io/wishlist/`
